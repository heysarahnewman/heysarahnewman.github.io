# heysarahnewman.github.io

Personal site — built with plain HTML & CSS, hosted on GitHub Pages.

## How to deploy

1. Create a new repo on GitHub named `heysarahnewman.github.io`
2. Push this folder's contents to the `main` branch
3. Go to **Settings → Pages** and set source to `main` branch
4. Your site will be live at `https://heysarahnewman.github.io`

## Structure

- `index.html` — the entire site (single page, no dependencies beyond Google Fonts)

## Editing

All content is in `index.html`. To update your "Now" section or add works, just edit the HTML directly. The CSS is embedded in the same file for simplicity.
